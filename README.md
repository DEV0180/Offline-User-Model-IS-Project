# Offline-User-Model-IS-Project:
# Prerequisites

Python: Version 3.7 or higher.

Hardware: mmWave Radar sensor (e.g., TI IWR series) connected via USB/Serial.

External Modules: The mmWave library (specifically the vitalsign module) must be present in your environment.

# Libraries and Dependencies

The system relies on the following Python packages:

pyserial: For serial communication with the radar hardware.

numpy: For numerical processing and signal arrays.

pandas: For data manipulation and CSV handling.

matplotlib: For real-time plotting and visualization.

scipy: For signal processing (peak detection).

tkinter: For the graphical user interface.
